# Лабораторная 5. Машинное обучение без учителя. Методы снижения размерности и задача кластеризации


## 1. Описание

В данной лабораторной работе изучить на практике методов data mining для снижения размерности данных и решения задач кластеризации с использованием Python (scikit-learn, umap-learn, joblib). 

Программа работает с датасетом Камни в почках

## 2. Запуск

Для запуска необходимо положить датасеты в папку проекта

Для запуска интерфейса - запустить файл main.py

## 3. Требования

Установите зависимости с помощью команды в корне всего проекта:
```text
pip install -r requirements.txt
```
