# Лабораторная 6.1. Работа со сложными данными. Обработка естественного языка и классификация текста

## 1. Описание

В данной лабораторной работе необходимо изучить на практике методов обработки текстовых данных и их векторизации.

Программа работает с песнями, определёнными в файле songs.txt

## 2. Запуск

Для запуска интерфейса - запустить файл main.py

## 3. Требования

Установите зависимости с помощью команды в корне всего проекта:
```text
pip install -r requirements.txt
```
