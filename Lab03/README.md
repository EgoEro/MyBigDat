# Лабораторная 3. Бинарная и многоклассовая классификация. Оценка качества задачи классификации

## 1. Описание

В данной лабораторной работе необходимо изучить и применить на практике методы бинарной классификации для решения задачи диагностики злокачественных новообразований на основе датасета Binary Classification with a Tabular Kidney Stone Prediction Dataset, diabetes visualization with seaborn а также выполнить оценку качества построенных моделей машинного обучения.


## 2. Запуск

Для запуска анализа датасета Binary Classification with a Tabular Kidney Stone Prediction Dataset необходимо запустить файл common.py.

Для запуска анализа датасета по диабету необходимо запустить файл self_work.py

## 3. Требования

Установите зависимости с помощью команды в корне всего проекта:
```text
pip install -r requirements.txt
```
